# Інтелектуальний особистий помічник для моніторингу зосередженості користувача

## Анотація
Проект являє собою програмно-апаратний комплекс для моніторингу та аналізу рівня зосередженості користувача під час роботи за комп'ютером у реальному часі. Система використовує методи комп'ютерного зору для відстеження міміки та рухів голови, розраховує ключові метрики уваги та генерує персоналізовані рекомендації за допомогою інтеграції з великими мовними моделями (LLM). Проект розроблено в рамках дипломної роботи (КПІ ім. Ігоря Сікорського).

## Основний функціонал
- **Комп'ютерний зір та біометрія:** Аналіз відеопотоку з веб-камери з використанням алгоритмів розпізнавання обличчя (MediaPipe Face Mesh).
- **Розрахунок метрик уваги (Real-time):**
  - **EAR (Eye Aspect Ratio):** Визначення ступеня відкритості очей для детекції моргання.
  - **PERCLOS (Percentage of Eye Closure):** Оцінка рівня втоми на основі відсотка часу із закритими очима.
  - **BPM (Blinks Per Minute):** Частота моргання.
  - **Head Pose Estimation (Pitch/Yaw):** Визначення напрямку погляду (робоча зона, відволікання, погляд униз).
- **Інтелектуальне реагування (AI-модуль):** Генерація контекстних порад для відновлення концентрації через Google Gemini API на основі зібраних метрик та часу доби.
- **Аналітичний модуль:** Збереження сирих даних сесії у форматі CSV з подальшою генерацією звітів та графіків (Pandas, Matplotlib) для порівняння експериментальних вибірок.
- **Адаптивний графічний інтерфейс:** Розроблений з використанням CustomTkinter, підтримує згортання в системний трей (Background Mode) для мінімізації відволікань.

## Архітектура системи та структура проекту
Проект побудовано за принципами модульної архітектури. Основні компоненти:

* `app.py` — Головний модуль графічного інтерфейсу користувача (GUI).
* `focus_core.py` — Ядро системи: математичний розрахунок метрик, фільтрація сигналів та кінцевий автомат станів уваги (NORMAL, WARNING, CRITICAL).
* `gemini_client.py` — Клієнт-обгортка для взаємодії з API Google Generative AI.
* `analytics.py` — Модуль постобробки даних, розрахунку ефективності та побудови діаграм.
* `csv_logger.py` — Система низькорівневого логування поточних метрик (з частотою 1 Гц).
* `tray_manager.py` — Модуль управління життєвим циклом програми у фоновому режимі (System Tray).
* `config.py` — Централізований файл конфігурації алгоритмічних порогів та гіперпараметрів.
* `/tests/` — Набір модульних тестів (Unit Tests) для перевірки надійності ядра системи.

## Технологічний стек
- **Мова програмування:** Python 3.10+
- **Комп'ютерний зір:** OpenCV, MediaPipe
- **Графічний інтерфейс:** CustomTkinter, Pillow
- **Обробка даних та аналітика:** Pandas, Matplotlib, NumPy
- **Штучний інтелект:** Google Generative AI SDK (Gemini Pro/Flash)
- **Тестування:** Unittest / Pytest

## Системні вимоги
- Операційна система: Windows 10/11, macOS, або Linux.
- Наявність підключеної веб-камери.
- Підключення до мережі Інтернет (для роботи модуля генерації AI-порад).

## Встановлення та налаштування

1. Клонування репозиторію:
   ```bash
   git clone [https://github.com/](https://github.com/)[ваш_нік]/kpi-focus-assistant.git
   cd kpi-focus-assistant

2. Створення та активація віртуального середовища:
   ```bash
   python -m venv venv310
   source venv310/bin/activate  # Для Linux/Mac
   venv310\Scripts\activate     # Для Windows

3. Встановлення залежностей:
   ```bash
   pip install -r requirements.txt

4. **Налаштування API (Опціонально):**
   Відкрийте файл `config.py` та замініть значення `gemini_api_key` на власний ключ від Google AI Studio.

## Запуск програми
- **Через вихідний код:** Виконайте команду `python app.py` у терміналі (попередньо активувавши віртуальне середовище).
- **Через виконуваний файл:** Запустіть файл `KPI_Focus_Assistant.exe` або скористайтеся пакетним файлом `StartApp.bat` для автоматичної ініціалізації.

## Етика та конфіденційність даних
Програмний комплекс розроблено з урахуванням принципів Privacy by Design. Обробка відеопотоку відбувається локально на пристрої користувача. Дані не записуються у відеоформаті та не передаються на сторонні сервери. До API штучного інтелекту надсилаються виключно знеособлені числові метрики (відсотки, частота моргання, час).

## Академічна цінність
Проєкт демонструє практичне застосування математичних моделей комп'ютерного зору для вирішення проблеми зниження когнітивної концентрації. Система забезпечує дослідників та користувачів інструментарієм для об'єктивної (числової) оцінки ефективності робочого процесу.
