# Інтелектуальний особистий помічник для моніторингу зосередженості користувача

## Опис проекту
Цей проект є дипломною роботою, що реалізує інтелектуальну систему моніторингу зосередженості користувача. Система використовує комп'ютерний зір для аналізу обличчя та очей через веб-камеру, розраховує метрики фокусу (PERCLOS, EAR, BPM) та надає персоналізовані поради через інтегрований ШІ (Google Gemini).

### Основні функції
- **Моніторинг у реальному часі**: Аналіз пози голови, моргання та зосередженості.
- **ШІ-поради**: Динамічні рекомендації для покращення продуктивності (можна ввімкнути/вимкнути).
- **Статистика**: Відображення часу фокусу, відволікань, коефіцієнту фокусу.
- **Мінімалістичний інтерфейс**: Світла тема, картковий дизайн (натхненний Pinterest).
- **Логування**: Збереження даних у CSV для подальшого аналізу.
- **Трей-режим**: Мінімізація в системний трей.

## Архітектура системи
- **focus_core.py**: Ядро аналізу (FocusAnalyzer клас).
- **app.py**: Графічний інтерфейс (CustomTkinter).
- **gemini_client.py**: Клієнт для ШІ (Google Gemini).
- **config.py**: Конфігурація параметрів.
- **csv_logger.py**: Логування сесій.
- **tray_manager.py**: Управління треєм.
- **text_render.py**: Рендеринг тексту на зображеннях.

## Вимоги до системи
- Python 3.10+
- Веб-камера
- Інтернет для ШІ (опціонально)

## Залежності
Встановіть залежності через pip:
```
pip install opencv-python mediapipe customtkinter plyer google-generativeai numpy scipy
```

## Запуск
1. Активуйте віртуальне середовище: `.\venv310\Scripts\activate`
2. Запустіть додаток: `python app.py`
3. Натисніть "Почати сесію" для початку моніторингу.
4. Використовуйте перемикач "Увімкнути AI Асистент" для активації ШІ-порад.

## Конфігурація
- Змініть `config.py` для налаштування порогів (наприклад, `thresh_normal`).
- Встановіть API-ключ Google Gemini у `config.py` (`gemini_api_key`).

## Тестування
Запустіть тести: `python -m pytest tests/ -v`

Доступні unit-тести для основних компонентів:
- `test_focus_core.py`: Тести для FocusAnalyzer (ініціалізація, розрахунки метрик, інтервенції).

## Етика та приватність
Система використовує камеру лише для аналізу обличчя. Дані не передаються третім сторонам. Перед використанням переконайтеся у згоді на моніторинг.

## Академічна цінність
Цей проект є дипломною роботою з теми «Інтелектуальний особистий помічник для моніторингу зосередженості користувача». Він демонструє:
- Інтеграцію комп'ютерного зору (MediaPipe) та ШІ (Google Gemini).
- Розробку ПЗ з використанням Python та сучасних фреймворків.
- Етичні аспекти моніторингу (згода користувача, приватність).

Для звіту дипломної роботи використовуйте код як основу реалізації, додавши теоретичну частину, діаграми та експериментальні результати.
