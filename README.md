# Інтелектуальний особистий помічник для моніторингу зосередженості користувача

## Анотація
Проєкт — дипломний програмно-апаратний комплекс для моніторингу та аналізу рівня зосередженості користувача під час роботи за комп'ютером. Система працює в реальному часі, аналізує відеопотік з веб-камери, обчислює метрики уваги та може генерувати короткі рекомендації через AI.

## Основний функціонал
- **Аналіз обличчя в реальному часі:** використання MediaPipe Face Mesh та OpenCV.
- **EAR (Eye Aspect Ratio):** визначення ступеня відкритості очей для детекції моргання.
- **PERCLOS:** оцінка рівня втоми за відсотком часу із закритими очима.
- **BPM:** частота моргань.
- **Оцінка голови та погляду:** класифікація зон уваги за положенням голови.
- **AI-поради:** інтеграція з Google Gemini для генерації дружніх порад.
- **Збереження даних:** логування метрик сесії у CSV файл.
- **GUI:** графічний інтерфейс на CustomTkinter із підтримкою системного трея.

## Структура проєкту
Основні файли і модулі:

- `app.py` — головний GUI-додаток.
- `main.py` — консольне виконання з відео та метриками.
- `focus_core.py` — обчислення EAR/PERCLOS/BPM, аналіз зони уваги, логіка станів.
- `config.py` — конфігурація порогів і налаштувань.
- `gemini_client.py` — обгортка для Google Generative AI.
- `csv_logger.py` — логування сесії в CSV.
- `tray_manager.py` — робота з системним треєм.
- `text_render.py` — рендер українського тексту поверх відео.
- `tests/test_focus_core.py` — модульні тести ядра.
- `requirements.txt` — список залежностей для запуску.
- `graphs/` — додаткові аналітичні скрипти та графіки.

## Технологічний стек
- **Мова:** Python 3.10+
- **CV:** OpenCV, MediaPipe
- **GUI:** CustomTkinter, Pillow
- **Аналітика:** Pandas, Matplotlib, NumPy
- **AI:** Google Generative AI (`google-generativeai`)
- **Повідомлення:** Plyer
- **Тестування:** Pytest / Unittest

## Системні вимоги
- Windows 10/11, macOS або Linux.
- Веб-камера.
- Інтернет для AI-модулю (опціонально).

## Встановлення та налаштування

1. Створіть і активуйте віртуальне середовище:
   ```bash
   python -m venv venv310
   source venv310/bin/activate  # Linux/Mac
   venv310\Scripts\activate    # Windows
   ```

2. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```

3. Якщо ви хочете використовувати AI-поради, у файлі `config.py` вкажіть свій `gemini_api_key`.

## Запуск програми
- Запуск GUI:
  ```bash
  python app.py
  ```
- Запуск консольної версії:
  ```bash
  python main.py
  ```

## Логування та аналітика
Сесійні дані зберігаються в каталозі `logs/` з унікальною міткою часу.

## Тестування
Запустіть тести командою:
```bash
python -m pytest -q
```
